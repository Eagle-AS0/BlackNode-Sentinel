# Contributing to BlackNode Sentinel

Thank you for considering contributing to BlackNode Sentinel 🚀

We welcome contributions that improve security, performance, documentation, and features.

---

## How to Contribute

1. Fork the repository
2. Clone your fork
3. Create a new branch:

   feature/your-feature-name

4. Make your changes
5. Commit clearly:

   feat: add runtime logging middleware

6. Push to your fork
7. Open a Pull Request

---

## Branch Rules

- Do NOT push directly to main
- Use feature branches
- Keep commits clean and meaningful

---

## Code Standards

- Follow clean architecture principles
- Write readable and maintainable code
- Add comments when needed
- Keep security best practices in mind

---

## Reporting Issues

If you find a bug:

- Open an issue
- Describe the problem clearly
- Add steps to reproduce
- Add screenshots if possible

---
